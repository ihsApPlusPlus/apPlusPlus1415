To run:
dowlnload a dasm compiler for the atari 2600
put the compiler in the same directory as the pong code
run a command prompt window from the directory of the code and run with this:
	dasm Pong.asm -f3 -oPong.bin
this will make a compiled .bin file to run
run in a atari 2600 emulator
